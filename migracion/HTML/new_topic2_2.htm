Para poder realizar la migración de Justicia a SAIDOJ el sistema valida que el campo que se ingresa de justicia Observaciones este correctamente diligenciado, de esta forma:

 

AÑO 2013 CAJA 0026 N 0470

 

El usuario debe ingresar:

- La palabra AÑO

-Espacio

-El año que es un campo numérico de 4 digitos

-Espacio

-La palabra CAJA

-Espacio

-El número de la caja que es campo numérico de 4 digitos y se debe rellenar con ceros hasta la izquierda para completar su longitud

-Espacio

-La palabra N

-Espacio

-El número del consecutivo que es campo numérico de 4 digitos y se debe rellenar con ceros hasta la izquierda para completar su longitud

 

El módulo de Procesos con inconsistencias permite validar y corregir los registros que quedaron mal ingresados en el campo de observaciones (Justicia Siglo XXI), para realizar estas correcciones debe realizar los siguientes pasos:

 

 

 

 

1. El usuario selecciona la opción del menú Procesos con Inconsistencias, el sistema presenta el siguiente formulario:

 

 

 

Procesos con Inconsistencias

 

Fecha Inicial : Campo de calendario donde se selecciona el rango de fecha Inicial desde el cual se va a realizar la consulta.

Fecha Final   : Campo de calendario donde se selecciona el rango de fecha Final hasta el cual se va a realizar la consulta.

Despacho: Lista de desplegable donde el usuario selecciona el despacho para realizar la consulta.
