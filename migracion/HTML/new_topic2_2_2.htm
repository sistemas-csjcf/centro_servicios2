Migrar Procesos de Justicia a SAIDOJ

Después de verificar que el campo observaciones este correctamente diligenciado, debe realizar los siguientes pasos (Ver Nota Punto 1):

 

 

 

 

1. El usuario selecciona la opción del menú Procesos a Migrar, el sistema presenta el siguiente formulario:

 

 

 

Migrar Procesos de Justicia a SAIDOJ

 

Fecha Inicial : Campo de calendario donde se selecciona el rango de fecha Inicial desde el cual se va a realizar la consulta.

Fecha Final   : Campo de calendario donde se selecciona el rango de fecha Final hasta el cual se va a realizar la consulta.

Despacho: Lista de desplegable donde el usuario selecciona el despacho para realizar la consulta.

 

 

 

 

 

2. El usuario ingresa los datos y selecciona el botón Consultar, el sistema presenta los procesos que cumplen con los criterios ingresados y que estan listos para migrar:

 

 

 

 

 

 

 

3. El usuario selecciona el Botón Migrar a SAIDOJ, que esta en la parte superior en inferior de la consulta, el sistema presenta el mensaje indicando que se realizo la actualización correctamente:

Nota:

 

1. Para poder realizar la migración de Justicia a SAIDOJ el sistema valida que el campo que se ingresa de justicia Observaciones este correctamente diligenciado, de esta forma:

 

AÑO 2013 CAJA 0026 N 0470

 

El usuario debe ingresar:

- La palabra AÑO

-Espacio

-El año que es un campo numérico de 4 digitos

-Espacio

-La palabra CAJA

-Espacio

-El número de la caja que es campo numérico de 4 digitos y se debe rellenar con ceros hasta la izquierda para completar su longitud

-Espacio

-La palabra N

-Espacio

-El número del consecutivo que es campo numérico de 4 digitos y se debe rellenar con ceros hasta la izquierda para completar su longitud
